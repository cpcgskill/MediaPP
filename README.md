# MediaPP
一个媒体处理工具， 使用ffmpeg+sox。
大部分处理代码来自与我之前处理蛇身体绑定插件教程的视频的代码，
本项目主要是将其整合成一个带GUI的工具，方便使用。


因为之前的都是以脚本的形式存在，所以后面应该会进行一些重构。
